# 🚀 CareerBoost - AI-Powered Career Development Platform

CareerBoost is a full-stack web application designed to help students, professionals and job seekers accelerate their career growth.

## ✨ Features
- 🎯 **Smart Job Matching** - AI-powered job recommendations based on skills
- 📄 **Resume Builder** - ATS-optimized resume creator with score calculator
- 🎤 **Mock Interviews** - Practice with AI feedback and scoring
- 👤 **Profile Management** - Skills, experience and job preferences

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Auth:** JWT Authentication

## 🚀 Getting Started

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
